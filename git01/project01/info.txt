git 이론
	git 버전관리시스템, github git을 관리할 수 있는 서버를 제공하는 사이트
git 구조
	local repository(로컬 저장소)
	staging area(스테이징 영역)(스테이징=커밋하기 전 git이 관리가능한 영역)
	working directory(작업 폴더)(작업 폴더를 만들었으면 반드시 init 할것)
